## About `ExpoGrowth`

v1.0

This interactive application allows you to explore population dynamics under
exponetial growth. Change the parameters to see what happens.

Check this application online: [ExpoGrowth]()

## Run this application on R Studio

To run this application localy, simple paste the following code on `R` console: 
```{r} 
 install.packages("shiny")
 shiny::runGitHub("ExpoGrowth", "paternogbc")
```
